Hello and welcome in this Documentation that will show you how to setup and manage the JMAX magazine theme.

Setup the theme

Install JMAX in 6 simple steps!

  1. Download the archived theme package. 
  2. Unpack the archive "jmax.zip". 
  3. You will find an other archive called uploadme.zip
  4. Go to worpress admin panel -> appearance -> themes -> Click the "Add new" button just beside the page title.
  5. Click upload theme button just beside the page title
  6. Select the archive called uploadme.zip and start uploading
  7. Activate the theme and enjoy!

 

Logo

After Installing The theme you can start by uploading your own logo, by following this simple steps:

  1. First login to your wordpress admin panel go to appearance.
  2. Go To theme options After finding theme options panel go to "General" section.
  3. You will find some options including "upload Your Logo".
  4. Click the blue add media button and select your logo from the media uploader, and than click the "Send to options tree" button to confirm and click "Save changes"

As Well as the login logo ,the favicon, you can upload them from the "General" Section on the theme options panel under appearance menu item!

Setup the slider

You can setup the slider by following this steps

  • Login to your wordpress admin panel
  • Under appearance menu find "Theme Options"
  • Under the theme options panel go to the "Home slider" Tab.
  • Select The Slider categories.
  • Select order by method ( Date, Title, Views, Comments or Random )
  • Select order method  ( Descending or Ascending )
  • Select Slider Number Of Posts ( between 3 and 15 )
  • Slideshow ? ( if you choose yes the slider will turn automatically )

Setup The Home Page

You can setup the home page by following this steps :

  • Login to your wordpress admin panel
  • Under appearance menu find "Theme Options"
  • Go to "HOME PAGE" Section.
  • Select the home page style, there are 4 styles that show the basic latest posts
  • If you choose Modular home page you will find bellow "home page moduls".
  • Click add new to add new moduls.
  • Each modul can be configured to display the posts that you want.

 

Customization

JMAX Wordpress theme comes with a rich options panel, that help the user to customize every element in the theme, to do this, just follow this steps:

  1. Login to the wordpress admin panel
  2. Under the menu item appearance you can access to the theme options panel
  3. You will find different tabs on the left of this panel go to the "Customization" Tab
  4. You will find many options it's for you to create your own skin.

Social Icons

JMAX Make it easy to display beautiful links to your social media profiles or pages, this links will be displayed automaticly on the top menu and the about widget, to do so, just follow this steps :

  • Login to the wordpress admin panel
  • Go to theme options panel
  • Go to The "Social Links" Tab
  • Fill your desired social media with your links

There are 11 social media icons available : Facebook, Twitter, Google +, Linkedin, Instagram, Youtube, Vine, Soundcloud, Flickr, Pinterest and Github, in adition to this there is an option to show or hide the RSS link of your site.

Category Settings

JMAX comes with a rich category features, slider, colors, backgrounds, and styles :

  • The category slider will be enabled automatically, if you want to hide it you can go to your wordpress admin panel -> appearance -> theme options -> Category Settings -> And from there you can disable the category slider.
  • The category color and background can be selected from the category edit page, you will find it under the posts menu item in your wordpress dashboard.
  • The category styles can be changed from your wordpress admin panel -> appearance -> theme options -> Category Settings -> Category layout, you will find 4 layouts.
